## テンプレート使用時の注意事項

1. すべてのセクションを埋める必要はありません。プロジェクトに応じて適宜調整してください。
2. クラス図はオプションですが、複雑な設計の場合は記載を推奨します。
3. 設計の検証セクションは、設計思想（変更容易性・可読性・単一責務）が実現されているかを確認するために重要です。
4. **全アイテムを統一的に扱える設計**を最優先してください。
