## テンプレート使用時の注意事項

1. 全てのGherkinシナリオがテストケースとして網羅されていることを確認してください。
2. テストケースは具体的に書いてください（具体的な数値を含める）。
3. クラス設計書で定義されたクラス/メソッドを前提としたテストを計画してください。
4. テストは失敗する（Red状態）ことを前提に計画してください（実装はフェーズ4）。
