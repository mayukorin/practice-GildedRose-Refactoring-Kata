# アイデア壁打ちディレクトリ

このディレクトリは、Gilded Roseプロジェクトにおけるスペック駆動開発とClaude活用に関するアイデアを整理・議論するための場所です。

## 目的

- スペック駆動開発（Specification-Driven Development）の実践方法を検討
- Claudeを活用した効率的な開発フローの設計
- リファクタリングと新機能開発を組み合わせた学習体験の構築

## 議論のトピック

### 1. スペック駆動開発とは？
- 仕様を形式化・構造化する
- 仕様から直接テストケースを導出
- テストファーストで実装を進める

### 2. Gilded Roseでの適用可能性
- 既存の要件仕様（GildedRoseRequirements.md）の活用
- レガシーコードのキャラクタライゼーション
- 新機能（Conjured items）の追加

### 3. Claudeの役割
- 仕様の分析と形式化の支援
- テストケース生成の自動化
- リファクタリングの提案とレビュー
- ドキュメント生成
